# Local Build Checklist — JARVIS ภาษาไทย

## เป้าหมาย

ใช้เช็กลำดับการติดตั้ง JARVIS ภาษาไทยบนเครื่องส่วนตัว โดยไม่อัปโหลด API key หรือข้อมูลลับขึ้น GitHub

## ขั้นตอน

1. แตกไฟล์โปรเจกต์ในเครื่องส่วนตัว
2. สร้าง virtual environment ด้วย Python 3.11 หรือ 3.12
3. ติดตั้ง dependency จากไฟล์ requirements ที่อยู่ในชุดดาวน์โหลด
4. สร้างไฟล์ `config/api_keys.json` จากไฟล์ตัวอย่าง
5. ใส่ Gemini API key ในเครื่องตัวเองเท่านั้น
6. รัน `sanity_check_th.py`
7. ทดสอบคำสั่งภาษาไทยระดับพื้นฐานก่อน
8. ทดสอบโหมดงาน วพอ. ด้วยข้อมูลจำลองก่อนใช้ข้อมูลจริง

## ห้ามอัปขึ้น GitHub

- `config/api_keys.json`
- `.env`
- cert/key/token ทุกชนิด
- ไฟล์ Excel หรือเอกสารที่มีข้อมูลนักเรียน/บุคลากรจริง
- cache, virtual environment, log runtime
